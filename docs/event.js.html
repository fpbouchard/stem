Events

Stem.Events is a mixin that allows objects to trigger events and allow other object to bind and unbind from them.

As is defined in mixins.js.coffee, the @implements class method should be used to implement the mixin:

class MyEventableClass
  @implements Stem.Events

Event Descriptors

The event descriptors used in bind and unbind are, simply put, comma-separated event names that are a shortcut to binding multiple events to the same handler. For example, given this handler:

handler = (context, newValue) -> console.log newValue

Calling this:

context.bind "change:mode, change:state", handler

Is the same as calling this:

context.bind "change:mode", handler
context.bind "change:state", handler
